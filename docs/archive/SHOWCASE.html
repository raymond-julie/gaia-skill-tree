✨ What Is GAIA?

GAIA maps AI agent capabilities as a composable skill tree. Every capability exists somewhere on the graph:

  • Skills start at the foundation tier (Basic skills)
  • Awaken through evidence and use
  • Evolve into higher ranks (Extra, Ultimate)
  • Fuse into capabilities greater than the sum of their parts

🎯 Why It Matters

○ Single Source of Truth

All agent capabilities live in one place. No more "Does Agent-X have web-search?" — the registry knows.

◇ Composability First

Skills know their dependencies. web-scrape + embed-text + store-vector = knowledge-harvest emerges automatically.

◆ Evidence-Based Versioning

Ranks reflect real performance (★ system), not declarations. Skills level up through evidence, not hope.

🔗 MCP Integration

Expose your skill tree natively to Claude, Cursor, and any MCP-compliant tool. Agents query capabilities automatically.

📊 Governance & Audit

Track skill versioning, evidence trails, and deployment history. Who approved this? When did it change? Why?

🚀 Scale Without Chaos

10 agents, 100 skills, 3 teams. GAIA keeps everything in sync. No merge conflicts. No redundant code.

🚀 Get Started Now

Step 1: Install

pip install gaia-cli

Step 2: Initialize

gaia init --user your-name

Step 3: Scan & visualize

gaia scan && gaia tree

Done! You now have a visual skill tree. Next: explore the registry.

gaia tree --canon
