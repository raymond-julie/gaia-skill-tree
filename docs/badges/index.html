README badges

Wear your Gaia rank.

Contributors who earn named skills can drop a Gaia-branded badge into their own repo READMEs. Three variants — highest rank, total skills, and the single-line @handle/skill · N★ identity badge — all regenerate on every gaia docs build, so the values track the live registry.

live

Variants

handle.svg
Single-line identity. @handle in honor red, top named skill in its rank color, star count in the same hue.
rank.svg
Highest rank earned. Right panel color is the rank tier (1★ cyan → 5/6★ gold).
skills.svg
Total skill count. Amber panel, always.
powered-by-gaia.svg
Generic fallback for non-contributors who use the registry.

Generate yours

Enter your Gaia handle. Badges live at gaia.tiongson.co/badges/<handle>/ and link back to your contributor profile at /u/<handle>/. The snippet bakes in ?repo=<owner/name> so the upcoming edge validator can flag misuse — badges embedded in a repo not on your links.github list render red.
