Github Badges

Contributors who earn named skills can drop a Gaia-branded badge into their own repo READMEs. These badges track your highest rank, total skills, or top skill in real-time, regenerating on every release.

Variants

      
Four distinct formats serving fallback, aggregate, and highly precise identity views.

      
      

        

          
          

            
handle.svg

            
Single-line identity. @handle in honor red, top named skill in its rank color, star count in the same hue.

          

        

        

          
          

            
rank.svg

            
Highest rank earned. Right panel color is the rank tier (1★ cyan → 5/6★ gold).

          

        

        

          
          

            
skills.svg

            
Total skill count. Amber panel, always.

          

        

        

          
          

            
powered-by-gaia.svg

            
Generic fallback for non-contributors who use the registry.

          

        

      

    

  


  

    Badges are static SVGs regenerated by gaia docs build and served from
    gaia.tiongson.co. GitHub's camo proxy caches images for ~24 hours, so newly earned
    ranks may take a day to propagate after a release. Logo: the Gaia diamond seal.
