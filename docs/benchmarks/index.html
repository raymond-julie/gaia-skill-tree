Benchmarks

Reproducible benchmark evidence for Gaia skills. Every score carries a reproducibility fingerprint (dataset hash + benchmark input hash), a provenance label (verifier-attested, CI-reproduced, mirrored, or pending), and an attestor identity. Self-attested scores are never accepted. Mirrored scores appear as citations only and are excluded from Trust Magnitude.
