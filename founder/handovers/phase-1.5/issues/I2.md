### `checkSourceTenureDaysGte180AorS()` implementation notes

- Compute post-weight magnitude per row first to determine effective A/S classification.
- For each row classified A or S, read `evidenceRow.sourceStartedAt` (ISO date, new from I1).
- Take the max source age across all A/S rows. Return `True` if max age ‚â•180 days.
- If no A/S rows exist, return `False` (not `None` ‚Äî a skill with zero A/S evidence cannot satisfy this).
- If `sourceStartedAt` is absent on an A/S row, treat the row as having age=0 days (conservative fallback).
