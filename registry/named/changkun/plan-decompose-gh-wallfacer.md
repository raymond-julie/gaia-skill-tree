## Overview

Wallfacer is an autonomous engineering platform that moves work through four stages in sequence: natural-language chat to surface intent, structured specs to pin scope, task decomposition to break work into atomic units, and code generation to close each task. The four stages form a pipeline — the output of each stage is the input to the next — so a vague request progressively crystallises into committed code without the user managing the handoffs manually.
