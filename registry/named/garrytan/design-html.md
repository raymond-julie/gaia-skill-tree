## Overview

Design to Production HTML translates a visual design or description directly into shippable, self-contained HTML/CSS. It handles real text reflow, computes element heights dynamically, and produces fully responsive layouts — with zero external dependencies beyond optional framework integration. Input can be a mockup image, a CEO plan, or a plain description.
