## Overview

CEO Plan Review interrogates a development plan from the founder's chair across four distinct modes. SCOPE EXPANSION dreams big; SELECTIVE EXPANSION cherry-picks high-leverage enhancements; HOLD SCOPE maximises rigor on the current plan; SCOPE REDUCTION strips to essentials. Each mode examines architecture, security, data flows, and edge cases, ensuring plans are fully stress-tested before a single line of code is written.
